Définitions

La page web est obtenue par interprétation d'un code HTML envoyé par le serveur en réponse à la requête qui lui est envoyée (adresse en http://www.etc).
C'est le navigateur qui interprète ce code.

Il existe deux sortes de balises :

  1. Les balises les plus courantes encadrent du code qui se termine pas un balise dite fermante. Exemple : <balise> contenu HTML </balise>
  2. Les balises dites orphelines qui n'encadrent aucun code et qui se ferment toutes seules. Exemple : <br/> <hr/> <imb/> <input/> etc...
